## 常用到的头文件
```cpp
#include <string>   //标准字符串类
#include <iostream> //标准输入输出
#include <vector>  //标准向量容器
#include <set>     //标准集合容器
#include <map>     //标准映射容器
#include <algorithm> //标准算法
using namespace std;    //打开标准库的名字空间
```
